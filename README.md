# ResolveFlow AI

ResolveFlow AI is an advanced Logistics Support Intelligence Platform designed to streamline operations, enhance customer support, and provide real-time visibility into global supply chains.

## Features

- **Global Command Center**: Real-time 2D network map visualizing active hubs, live shipment traffic, and AI-predicted risk zones.
- **AI-Powered Inbox**: Smart triage console with automated ticket routing, customer context panels, and SLA countdowns.
- **Integrated Dashboard**: High-level KPIs, delay distribution charts, and live ops feed.
- **Intelligent Workflows**: Visual automation builder for triggering support actions based on real-time events.
- **Context-Aware Assistant**: Built-in AI helper that understands which page you are on and answers targeted operational queries.

## Tech Stack

- **HTML5** & **Vanilla JavaScript** (Zero framework dependencies)
- **CSS3** (Custom design system, flexbox/grid layouts, CSS variables)
- **Chart.js v4** (Data visualization)
- **SVG / SMIL** (Animated logistics network maps)

## How to Run

1. Clone or download the repository.
2. Open `index.html` directly in your modern web browser (Chrome, Firefox, Safari, Edge).
3. No build step or local server is strictly required, though using Live Server (VS Code) is recommended for optimal performance.

## Design Philosophy

- **Dark Theme First**: High contrast, low eye-strain interface built for operations teams staring at screens for hours.
- **Glassmorphism & Depth**: Subtle blurs, transparent panels, and elevated shadows create a modern hierarchy.
- **Information Density**: Maximizing actionable data per square inch without feeling cluttered.
- **Motion as Data**: Animations (pulsing hubs, moving particles) aren't just for show; they instantly convey network health and activity states.
